# Minesweeper Field

## Prerequisites
- Visual Studio with C++ 20 Standard support.
- Desktop Development with C++ package via Visual Studio Installer.

## Step by Step Guide

1. Open *Minesweeper Game.sln* file.
2. Press *Local Windows Debugger* in Visual Studio.
